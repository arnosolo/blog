## 项目经验

### AiPassportPhoto

[这个网站](http://aipassportphoto.com/)去除照片背景获得白底照片. 我做的是前端的部分. 这个项目的主要使用的技术有:

- 使用 Vite / Vue3 / Ts / Unocss 构建

- 使用 Github Action 实现**自动部署**

- 使用 vue-i18n 实现**多语言**

- 使用 Aws Cloudfront 和 Aws S3 实现**全球CDN**

- 使用 vite-ssg 实现**静态站点生成(SSG)**
